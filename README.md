本项目只用来提交MofficeRN项目zip发版包

外网域名+项目名路径+包名
例:
https://xxx.haier.com/haier/wuliu/btb/test/ios/btb_test_ios_2018101801.zip

红色部分:https的外网域名
蓝色部分：路径名，能提现出产品，环境，系统名称
包名：使用产品、环境、系统、日期断线链接，20181018后面的01是小版本号，建议保留。

注意：
1、	域名必须是https的，否则ios环境有时会不成功
2、	包名要体现出项目，环境，系统，日期
3、	包名必须是zip
4、	包内不能包含无用的文件夹，必须是直接到js bundle相关文件
5、	环境区分出测试(test)和生产(prod)两个环境
6、	Js主文件名必须是main.jsbundle
